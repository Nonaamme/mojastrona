# Barber Shop Booking Website

Simple web application developed as part of a university course.  
The project demonstrates basic full-stack development with a React frontend and a PHP backend connected to a MySQL database.

## Features
- Responsive interface for booking appointments
- Basic CRUD operations for managing bookings
- Data persistence with MySQL
- API communication between React and PHP backend

## Technologies
- React.js
- PHP
- MySQL
- HTML5 / CSS3

## Context
This project was created as part of a university assignment to practice building a small full-stack application.
